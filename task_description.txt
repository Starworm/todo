- Создание компонента списка задач (TodoListComponent)
    Создайте компонент, который будет отображать список задач.
    Данные о задачах можно сохранить в виде массива внутри компонента.
    Каждая задача должна отображаться как элемент списка с текстом задачи и
      чекбоксом для отметки выполнения.
- Добавление новой задачи
    Реализуйте форму для ввода текста новой задачи.
    При отправке формы добавляйте новую задачу в список.
    Очищайте поле ввода после добавления задачи.
- Отметка задачи как выполненной
    При клике на чекбокс рядом с задачей, задача должна отмечаться как
    выполненная (или невыполненная, если была выполненной).
    Выполненные задачи должны быть визуально отмечены (например,
      зачеркнутым текстом или другим стилем).
- Удаление задачи
    Добавьте кнопку или ссылку для удаления каждой задачи.
    При клике на кнопку удаления соответствующая задача должна быть удалена
      из списка.
- Стилизация приложения
    Используйте Angular Material или Bootstrap для стилизации компонентов.
    Примените соответствующие стили для создания привлекательного и
      удобного пользовательского интерфейса.
- Тестирование (опционально)
    Напишите базовые юнит-тесты для компонента TodoListComponent с
      использованием Angular Testing Framework.
    Протестируйте методы добавления, удаления и отметки задачи как
      выполненной.
